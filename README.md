
프로젝트 명  : ArtAuction

예술가들을 위한 경매시스템 기반 웹 오픈마켓

1. 기획의도
- 개인예술가들의 작품 공유 플랫폼의 부족
- 소규모 개인 예술가들이 작품을 쉽게 판매/공유 할 수 있는 오픈마켓 페이지 개발 요구 
- 예술가들은 작품의 가치를 스스로 책정하고 경매를 통해 보상을 받음
